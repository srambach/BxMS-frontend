Imports

External Data Objects are Data Objects not explicitly defined within a Project or Project's dependencies that a rule author may need available. They are usually provided by the Java runtime. For example: java.util.List

Configuration Files

Whenever a runtime component (such as jbpm console) is about to process kjar it looks up kmodule.xml to build its runtime representation. In addition to kmodule.xml a deployment descriptor (that provides fine grained control over deployments) is available (since 6.1).
